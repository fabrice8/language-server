# Marko VSCode

Marko tooling for VS Code, powered by [marko-language-server](../../tree/master/server)

# Features

- [x] Go To Definition
- [x] Syntax HighLighting
- [x] Autocompletion
- [x] Tag Matching
- [x] Error reporting
- [ ] Linting
- [ ] Debugging
